## 技术点

* 前端：vue-cil + vue2 + vuex + vue-router

* 服务端：express + sequelize

* 服务器环境：ubuntu14.04 + nginx + mysql + nodejs + pm2

## Installation

``` bash
# install dependencies
npm install

# serve at localhost:9001
npm start
```
